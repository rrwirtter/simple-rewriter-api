STEP 1: Create GitHub Repository
─────────────────────────────────

1. Go to: https://github.com
2. Click "New Repository" (green button)
3. Name: simple-rewriter-api
4. Select: Public
5. Click "Create repository"

STEP 2: Upload ALL Files
─────────────────────────

1. In your repository, click "uploading an existing file"
2. Go to: I:\AI To Human\Simple Re-Writter\api-deployment\
3. SELECT ALL FILES (Ctrl+A)
4. DRAG ALL FILES to GitHub
5. Click "Commit changes"

IMPORTANT: Upload ALL files at once! No subfolders needed!

STEP 3: Deploy to Vercel
─────────────────────────

1. Go to: https://vercel.com
2. Click "Sign Up" → Use GitHub
3. Click "Add New" → "Project"
4. Select "simple-rewriter-api"
5. Click "Deploy"
6. Wait 1-2 minutes ⏳
7. DONE! 🎉
